# Dashboard-on-amazon-prime-vdo
**Dashboard Created on Amazon Prime Video – Small Description**  A **dashboard created on Amazon Prime Video** is a visual interface that helps users easily access and manage movies, TV shows, and other content. 
Tech stack : Power bi desktop - Main data processing platform used to create the report
kaggle : used Dataset
Data Source : kaggle, chrome
**Features and Highlights of the Dashboard on Amazon Prime Video**

**Features:**

* Personalized movie and show recommendations
* Search and filter options
* Watchlist to save content
* Continue watching section
* Content categories (action, comedy, drama)

**Highlights:**

* Simple and user-friendly interface
* Attractive thumbnails and banners
* Trending and popular content section
* Easy navigation for quick access 🎬📺
